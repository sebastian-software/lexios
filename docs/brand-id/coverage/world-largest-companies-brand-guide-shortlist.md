# Shortlist: Useful Brand-Guide Candidates

Stand: 2026-04-13

## Why This Exists

The full matrix is intentionally broad. It keeps any company where the first automated pass found official public material that looked at least somewhat brand-relevant.

For actual `Brand ID` work, that is still too loose.

This shortlist narrows the pool to companies where the evidence URL itself points at something more brand-adjacent than generic press or investor content, such as:

- `brand`
- `logos`
- `design`
- `logo`
- `identity`
- `style`

It also keeps the few rows classified as `rich_brand`.
